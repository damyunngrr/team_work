 Password: <input type="password" name="password"><br><br>
 <input type="submit" value="Login">

Aim: To demonstrate SQL Injection vulnerability in a PHP login form.

Vulnerable Code:

<?php
$host = "localhost";
$user = "root";
$pass = "";
$db = "testdb";

// Connect to database
$conn = mysqli_connect($host, $user, $pass, $db);
if (!$conn) {
    die("Connection failed: ". mysqli_connect_error());
}

if (isset($_POST['login'])) {
    $username = $_POST['username'];
    $password = $_POST['password'];

    // Vulnerable SQL query (no validation)
    $sql = "SELECT * FROM users WHERE username = '$username' AND password = '$password'";
    $result = mysqli_query($conn, $sql);

    if (mysqli_num_rows($result) > 0) {
        echo "<h3>Login Successful</h3>";
    }
    else {
        echo "<h3>Invalid Login</h3>";
    }
}
?>

Secure Version:

<?php
$host = "localhost";
$user = "root";
$pass = "";
$db = "testdb";

$conn = new mysqli($host, $user, $pass, $db);
if ($conn->connect_error) {
    die("Connection failed: ". $conn->connect_error);
}

if (isset($_POST['login'])) {
    $username = $_POST['username'];
    $password = $_POST['password'];
    
    // Secure query with prepared statement
    $stmt = $conn->prepare("SELECT * FROM users WHERE username = ? AND password = ?");
    $stmt->bind_param("ss", $username, $password);
    $stmt->execute();
    $result = $stmt->get_result();

    if ($result->num_rows > 0) {
        echo "<h3>Login Successful</h3>";
    } else {
        echo "<h3>Invalid Login</h3>";
    }
    $stmt->close();
}
$conn->close();
?>

7. Demonstrate Encryption Using Python

Aim: To encrypt and decrypt text using Python's Fernet module.

Code:

from cryptography.fernet import Fernet

# Generate a new Fernet key (save this key for decryption)
key = Fernet.generate_key()
cipher = Fernet(key)

# Display the encryption key
print("Generated Encryption Key:")
print(key.decode())
print("\nNote: Save this key to decrypt the message later.\n")

# Get plaintext input from the user
plaintext = input("Enter the text you want to encrypt: ")

# Encrypt the plaintext
encrypted_text = cipher.encrypt(plaintext.encode())
print("\nEncrypted Text:")
print(encrypted_text.decode())

# Decrypt the ciphertext
decrypted_text = cipher.decrypt(encrypted_text).decode()
print("\nDecrypted Text:")
print(decrypted_text)

3. Factorial of a number

Aim: To write a shell program to calculate the factorial of a number.

Code:

#!/bin/bash

read -p "Enter a positive integer: " num

if [ "$num" -lt 0 ]; then
    echo "Factorial not defined for negative numbers."
    exit 1
fi

fact=1
i=1

while [ $i -le $num ]
do
    fact=$((fact * i))
    i=$((i + 1))
done

echo "Factorial of $num is: $fact"

Output:

5. Convert a String to Hash and Verify using Python

Aim: To create and verify hashes of user input.

Code:

import hashlib

# Create hash
text = input("Enter text: ")
hashed = hashlib.sha256(text.encode()).hexdigest()
with open("hash.txt", "w") as file:
    file.write(hashed)

# Verify hash
check = input("Enter text again to verify: ")
check_hash = hashlib.sha256(check.encode()).hexdigest()
with open("hash.txt", "r") as file:
    saved_hash = file.read()

if check_hash == saved_hash:
    print("Match found!")
else:
    print("No match")
